Routes

User authentication:
    /signup: POST - Create a new user account
    /login:  POST - Login to an existing account.[aaashar card number + password]

Voting:
    /candidates: GET - Get the list of candidates
    /vote/:candidates: POST - Vote for a specific candidates

Vote Counts:
    /candidates: GET - /vote/counts: GET - Get the list of candidates sorted by their vote counts

User Profiles:
    /profile: GET - Get the user's profile information
    /profile/password: PUT - change the user's password

Admin candidate Management:
    /candidate: POST - Create new candidate
    /candidate/:candidateId: PUT - Update an existing candidate
    /candidate/:candidateId: DELETE - Delete a candidate fromt the list.
